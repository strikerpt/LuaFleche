# LuaFleche
Creation d'une fleche pointant vers le haut en fonction de la taille specifiee par l'utilisateur.

# Installation
Il faut executer le script:
```
./install.sh
```
# Utilisation
Il faut faire:

```
./luaFleche taile_de_la_fleche
```
# Demonstration
Il faut juste faire tourner:

```
./start.sh
```
