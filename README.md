# LuaFleche
Création d'une flèche pointant vers le haut en fonction de la taille specifiée par l'utilisateur.

# Installation
Il faut exécuter le script:
```
./install.sh
```
# Utilisation
Il faut faire:

```
./luaFleche taile_de_la_fleche
```
# Démonstration
Il faut exécuter:

```
./start.sh
```
